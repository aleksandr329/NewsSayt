Here is a tutorial project for SkillFactory to create a "News Portal" website.


Here, according to Django standards, we simply created and looked at what functionality this framework had. Also here we connected to Redis via Celery.

The accounts, news, subscriptions folders are applications.
The static folder contains all static css files.
All HTML templates are in the templates folder.
The database was used by SQLite.
